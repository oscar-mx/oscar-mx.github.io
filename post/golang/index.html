Golang及Goland安装

2019-12-02 golang

写在开头

Golang（简称Go）语言号称兼具 Python 等动态语言的开发速度和 C/C++ 等编译型语言的性能与安全性。本文主要讲述Golang的安装，环境配置，以及Goland的安装使用

Go运行环境安装与配置

1️⃣ 下载Go语言
国内直接到这个地址下载：
然后选择符合自己操作系统的安装包进行下载，我这里是win10为例：

2️⃣ Go语言的安装
下载好安装包后双击打开，然后就是一路无脑 Next😂，当然记得自己的安装目录，因为后面的 Golang的环境配置，会用到这个安装目录的地址，下面是安装完成后的界面：

3️⃣ Go语言的环境变量配置（这里以我的安装位置为E:\Go举例）
win10配置环境需要有五个变量
系统变量：GOROOT、Path
用户变量：GOPATH、GOBIN、Path（或者一个名称为PATH的）
右键我的电脑，选择属性后，再点击高级系统设置，如下图：

顶部选择 高级，然后点击环境变量，如下图：

点击进入后，可以看到用户变量和系统变量，在下方的系统变量中，点击新建GOROOT、Path两个变量，用户变量名为：GOPATH，变量值为 在go安装目录中新建一个名称位project的文件夹，进入后新建三个文件夹，分别是src、bin、pkg这三个的文件夹的作用 ：src（存放项目源码）bin（存放编译后生成的可执行文件）pkg（存放编译后生成的包文件），如下图：

用户变量名为：GOBIN，变量值为之前建好的project目录下的bin目录，如下图：

用户变量名为：Path，变量值为安装目录下边的bin目录，如下图：

系统变量名为：GOROOT，变量值为 你的Go语言安装的位置，如下图：

系统变量名为：Path，变量值为安装目录下边的bin目录，如下图：

4️⃣ 最后检查Go环境变量配置是否成功
Win+R 打开 运行 的窗口，输入cmd 或者直接打开 命令行提示符,出现小黑框,
先输入 go version,检查Go语言是否安装成功，
再输入 go env 检查环境变量是否配置成功，如下图三个红线，才是正确的
